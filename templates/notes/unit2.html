1. 什麼是運算子與運算元？

  • 運算子（Operators） 是指用來執行計算或比較的符號，例如：+, -, *, /, >, <, = 等。
  • 運算元（Operands） 是指運算子操作的對象，可以是數字、變數或運算式。

範例

x = 10 + 5  # 10 和 5 是運算元，+ 是運算子

2. 算術運算子

Python 中的基本算術運算子包括：

運算子 功能 範例 結果
+ 加法 5 + 2 7
- 減法 5 - 2 3
* 乘法 5 * 2 10
/ 除法 5 / 2 2.5
// 整數除法 5 // 2 2
% 餘數 5 % 2 1
** 次方 5 ** 2 25

範例

a = 10
b = 3
print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.3333
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000

3. 比較運算子

比較運算子用來比較兩個值，返回 True 或 False。

運算子 說明 範例
== 等於 5 == 5 (True)
!= 不等於 5 != 3 (True)
> 大於 5 > 3 (True)
< 小於 5 < 3 (False)
>= 大於或等於 5 >= 5 (True)
<= 小於或等於 3 <= 5 (True)

範例

print(10 == 10)  # True
print(10 != 5)   # True
print(10 > 5)    # True

4. 邏輯運算子

邏輯運算子用來將多個條件進行組合，返回布林值。

運算子 功能 範例
and 且 (5 > 3) and (5 < 10) (True)
or 或 (5 > 10) or (5 < 10) (True)
not 非 not(5 > 10) (True)

範例

print((5 > 3) and (5 < 10))  # True
print((5 > 10) or (5 < 10))  # True
print(not(5 > 10))           # True

5. 賦值運算子

用於將值賦予變數。

運算子 說明 範例
= 賦值 x = 5
+= 加後賦值 x += 3 相當於 x = x + 3
-= 減後賦值 x -= 3 相當於 x = x - 3
*= 乘後賦值 x *= 3 相當於 x = x * 3

範例

x = 10
x += 5  # x 現在是 15
x *= 2  # x 現在是 30
print(x)  # 30

6. 練習題目

題目 1：計算 BMI

讓使用者輸入身高（公尺）與體重（公斤），然後計算並顯示 BMI 值。

範例

輸入身高 (m)：1.70
輸入體重 (kg)：65
你的BMI是 22.49

範例程式

height = float(input("輸入身高 (m)："))
weight = float(input("輸入體重 (kg)："))
bmi = weight / height**2
print(f"你的BMI是 {bmi:.2f}")

題目 2：溫度轉換

將攝氏溫度轉換成華氏溫度，公式：華氏 = 攝氏 × 9/5 + 32

範例程式

celsius = float(input("請輸入攝氏溫度："))
fahrenheit = celsius * 9/5 + 32
print(f"華氏溫度為 {fahrenheit:.2f}")

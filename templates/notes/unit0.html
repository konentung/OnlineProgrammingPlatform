1. 變數與賦值

變數（Variable）是用來儲存數據的名稱，我們可以用 = 來賦值（assign value）。

範例


    x = 10  # 將數值 10 存入變數 x
    name = "Python"  # 將字串 "Python" 存入變數 name
    print(x)
    print(name)

2. Python的基本資料型態

類型 說明 範例
整數 (int)整數數值10, -5
浮點數 (float)小數數值3.14, -0.5
布林值 (bool)True / FalseTrue, False
字串 (str)文字資料'Hello', "Python"
列表 (list)有序可變集合[1, 2, 3]

3. 輸入與輸出

輸出 print()


    print("Hello, world!")
    print("變數 x 的值是：", x)

輸入 input()


    name = input("請輸入你的名字：")
    print("你好,", name)

4. 註解與程式碼可讀性

註解（Comment）可以幫助我們理解程式，但不會影響程式執行。

單行註解


    # 這是一個註解
    x = 10  # 這也是註解

多行註解


    """
    這是一個多行註解
    可以用來說明較長的程式邏輯
    """

5. 基礎練習題

題目 1：變數與型別檢查


    a = 10
    b = 3.14
    c = "Hello"
    print("變數 a 的值是", a, "，型別是", type(a))
    print("變數 b 的值是", b, "，型別是", type(b))
    print("變數 c 的值是", c, "，型別是", type(c))

題目 2：輸入與運算


    num1 = int(input("請輸入第一個數字："))
    num2 = int(input("請輸入第二個數字："))
    sum_result = num1 + num2
    print("它們的總和是", sum_result)

Python程式學習網站：第四單元「邏輯控制 (Control Flow)」

單元目標

本單元將介紹 Python 中的邏輯控制（Control Flow），包含以下主題：

  1. 條件判斷 (if、elif、else)
  2. 巢狀條件判斷
  3. 三元運算子
  4. 邏輯運算 (and、or、not)
  5. 應用練習題

1. 條件判斷 (if、elif、else)

條件判斷用於根據不同條件執行對應的程式碼。

範例

age = int(input("請輸入你的年齡："))
if age < 18:
    print("你是未成年人。")
elif age < 65:
    print("你是成年人。")
else:
    print("你是高齡者。")

2. 巢狀條件判斷

條件判斷可以互相嵌套，形成巢狀結構。

範例

score = int(input("請輸入你的成績："))
if score >= 60:
    if score >= 90:
        print("優秀！")
    else:
        print("及格！")
else:
    print("不及格！")

3. 三元運算子

三元運算子可用來簡化條件判斷。

範例

num = int(input("請輸入一個數字："))
result = "正數" if num > 0 else "非正數"
print(result)

4. 邏輯運算 (and、or、not)

邏輯運算子用於組合多個條件。

範例

x = int(input("請輸入一個數字："))
if x > 0 and x % 2 == 0:
    print("正偶數")
elif x > 0 and x % 2 != 0:
    print("正奇數")
else:
    print("非正數")

5. 應用練習題

題目 1：成績判斷

讓使用者輸入成績，並根據分數顯示評價。

範例輸出

輸入分數：85
你的等級是 B

範例程式

score = int(input("輸入分數："))
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
print(f"你的等級是 {grade}")

題目 2：身分驗證

請讓使用者輸入年齡與是否持有駕照，判斷是否可以開車。

範例程式

age = int(input("請輸入年齡："))
has_license = input("是否有駕照？(y/n): ").lower() == 'y'
if age >= 18 and has_license:
    print("你可以開車。")
else:
    print("你不能開車。")

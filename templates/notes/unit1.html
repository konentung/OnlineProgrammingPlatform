Python程式學習網站：第一單元「相等性」

單元目標

本單元將介紹 Python 中的相等性（Equality），包括：

  1. 基礎相等比較 (== vs !=)
  2. 數值與布林值比較
  3. 字串相等比較
  4. 物件相等性 (is vs ==)
  5. 列表與字典的相等性
  6. 練習題目

1. 什麼是相等性 (Equality)？

在Python中，相等性是指兩個值或物件是否相等。

  • ==：判斷兩個值是否相等
  • !=：判斷兩個值是否不相等

範例

print(5 == 5)  # True
print(5 != 3)  # True
print("hello" == "hello")  # True
print("hello" != "world")  # True

2. 數值與布林值比較

print(10 == 10.0)  # True
print(10 == 10.1)  # False
print(True == 1)  # True
print(False == 0)  # True
print(True == 0)  # False

3. 字串相等比較

print("Python" == "python")  # False (大小寫不同)
print("apple" == "apple")  # True
print("hello" != "world")  # True

可以使用 .lower() 來忽略大小寫：

print("Python".lower() == "python".lower())  # True

4. is vs ==

== 用於比較值是否相等，而 is 用於比較物件是否是同一個記憶體位置。

a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)  # True
print(a is b)  # False
print(a is c)  # True

5. 列表與字典的相等性

list1 = [1, 2, 3]
list2 = [1, 2, 3]
list3 = list1

print(list1 == list2)  # True
print(list1 is list2)  # False
print(list1 is list3)  # True

6. 練習題目

題目 1：數字與布林值比較

num = int(input("請輸入一個數字："))
if num == True:
    print(f"{num} 等於 True")
elif num == False:
    print(f"{num} 等於 False")
else:
    print(f"{num} 既不等於 True 也不等於 False")

題目 2：大小寫無關的字串比較

def case_insensitive_compare(str1, str2):
    return str1.lower() == str2.lower()

print(case_insensitive_compare("Python", "python"))  # True
print(case_insensitive_compare("Hello", "World"))  # False
